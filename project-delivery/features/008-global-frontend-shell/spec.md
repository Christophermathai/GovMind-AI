# Specification: Global Frontend Shell

## 1. Description
A unified, production-grade frontend shell for GovMind AI that implements the "Industrial Intelligence Console" aesthetic. It provides global navigation and the main dashboard for accessing all sub-modules.

## 2. Functional Requirements
- Global Sidebar Navigation mapped to core modules (Citizen, Business, Rights).
- Main Command Center dashboard with an "Omnibar" for semantic search.
- Dark mode default with high-contrast, sharp 1px borders.

## 3. Non-Functional Requirements
- Strictly follow `@frontend-design` aesthetics: no generic AI UI, no soft shadows, no glassmorphism.
- Typography: `Bricolage Grotesque` for headings, `JetBrains Mono` for utility text.
